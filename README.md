# VIOLA PROXY (FOR VERCEL)
DISCLAIMER! IF NOT USING IN VERCEL, PLEASE USE THE ORIGINAL VIOLA PROXY!

Viola is a lightweight web proxy (with many planned updates) designed to provide fast, reliable web access in restricted environments. It is primarily developed for integration with the COSMOSIS platform, allowing users to reach web resources while maintaining a simple, flexible deployment structure. Viola retrieves and serves web content through a controlled gateway, acting as a secure intermediary between users and the internet. It is intentionally minimal, adaptable, and easy to deploy across a variety of hosting platforms. Although Viola is built with COSMOSIS integration in mind, it can also be used as a standalone proxy for development, research, or educational purposes.
